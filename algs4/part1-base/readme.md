# 第一章

## 背包、队列和栈

### 背包

1.  只进行元素的添加，不进行删除操作
2.  对数据结构进行求和，平均值，平方差等操作

### 队列

1.  先进先出
2.  push 添加到最后一项 shift 移除第一项

### 栈

2.  后进先出
3.  push 添加到最后一项， pop 移出最后一项

## 链表

链表是一种递归的数据结构，它或者为空，或者指向一个节点的引用，该节点含有一个元素和一个指向另一条链表的引用
